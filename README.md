# Career FIT OS

> 나와 이 회사가 맞는지 빠르게 판단해서 시간을 아끼는 이직 판단 도구.

공고(JD)를 넣으면 — 핵심 기준 체크 → 판정 → 어필 포인트 → 이력서 초안까지 한 번에 나온다.
이력서는 한 번만 입력하고, 그다음부터는 공고만 넣으면 된다.

**Live:** https://career-fit-os.vercel.app/

---

## 철학

- 나도 회사를 고른다
- 서로 맞아야 오래간다
- 최종 결정은 내가 한다

판정이 "패스"여도 사용자가 "그래도 지원"을 선택할 수 있다.
OS는 판단을 돕고, 결정을 막지 않는다.

---

## 무엇을 하나

| 탭 | 역할 |
|----|------|
| 프로필 | 내 경력 + 판단 기준 설정 (한 번만) |
| 공고 판단 | 공고 넣으면 판정 + 어필 포인트 + 이력서 초안 자동 생성 |
| 지원 현황 | 분석한 공고 상태 관리 (지원 전 / 서류 / 면접 / 최종 / 패스) |

**핵심 기준 3개** — 직무 적합 / 환경 적합 / 성장 적합
**판정** — 지원 추천 / 검토 필요 / 패스

## 기술 스택

- localStorage 데이터 유지
